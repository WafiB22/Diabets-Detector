# Diabets-Detector
🩺 Diabetes Prediction using Machine Learning
📌 Overview

This project aims to predict whether a patient is diabetic or not (0 = non-diabetic, 1 = diabetic) using machine learning techniques based on medical data.

⚙️ Technologies
Python
Pandas
Scikit-learn
Streamlit
Joblib
🧠 Model

A classification model was trained on medical features such as glucose level, BMI, age, and insulin to predict diabetes outcomes.

📊 Evaluation

The model performance was evaluated using:

Accuracy
Confusion Matrix
💻 Application

An interactive web application was developed using Streamlit, allowing users to input patient data and receive real-time predictions.

🚀 How to Run
pip install -r requirements.txt
streamlit run dashboard_diabete.py

🎯 Objective

The goal of this project is to provide a simple and efficient tool for early diabetes detection using machine learning.
